1. 在管理后台添加服务器

在"服务器"标签页输入名称后点击"添加"，会生成一条唯一的安装命令。

2. 在目标服务器上运行

SSH 登录你的 VPS 后粘贴执行（需要 root 权限）：

curl -sL https://your-worker.workers.dev/install.sh | bash -s install <ID> <SECRET> <WORKER_URL>

支持 Ubuntu / Debian / CentOS / Rocky / AlmaLinux

3. 自定义上报间隔

在命令末尾追加间隔秒数（默认 60）：

... | bash -s install <ID> <SECRET> <WORKER_URL> 30

4. 卸载探针

curl -sL https://your-worker.workers.dev/install.sh | bash -s uninstall
